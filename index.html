DeepseekNPC

一个围绕玩家展开的单人陪伴型 Terraria 模组。DeepSeek 负责聊天、记忆、离线 Wiki、自动召唤剑和战斗联动，花精灵负责药草园，回声鲸负责采集、钓鱼与捡物，把陪伴感和工具感真正捏在一起。

聊天、记忆、草药园、采集钓鱼和战斗提示各自分工，又都围绕同一个单人世界运转。

它不是只加几个 NPC 的装饰向内容，而是一整套陪伴系统：有人陪你说话、有人替你盯药草园、有人替你跑采集流程，还有一组把查资料、战斗提示、测试工具接起来的辅助功能。

核心角色与伙伴

整个模组的手感，基本都落在这三位身上。一个负责“对话与信息”，一个负责“生活管理”，一个负责“移动采集与拾取”。

城镇 NPC / 对话核心

DeepSeek

  • 入住后提供闲聊、聊天、连接检查和人物设定入口。
  • 对话会积累聊天历史、长期记忆总结，以及她对玩家的关系印象。
  • 内置离线 Terraria Wiki 查询，可查配方、掉落、Boss、宠物、坐骑、饰品、钓鱼和物品用途。
  • 附近出现敌怪时会自动召唤一把剑协助攻击，也能把辅助攻击、Boss 指示、胜利称赞和鼠标皮肤这些联动能力串起来。
城镇宠物 / 药草园管家

花精灵

  • 可以抚摸，也可以通过交互菜单进入工具管理和收获袋。
  • 先交给她再生法杖或再生之斧，再带她去划定药草园区域。
  • 她会自动巡园、收成熟草药、补种同类种子，并把收获放进自己的储物空间。
  • 支持暂停、继续和结束任务，做成更像“正在照看”而不是单次脚本触发。
宠物 / 坐骑 / 采集伙伴

回声鲸

  • 右键打开功能 UI，空闲且没有 Boss 时左键可骑乘。
  • 支持目标物块采掘、定点钓鱼和背包收纳，把工具使用感做成伙伴行为。
  • 会自动吸附附近掉落物，并集中叼住后统一送回玩家（骑乘时也会触发吸附功能）。
  • 拾取节奏覆盖普通物品，也照顾到心、魔力星等即时拾取物，整体体验更顺。

主要系统

除了角色本身，这个模组真正耐玩的地方在于一堆“围绕玩家工作的系统层”。它们不是独立小插件，而是互相搭着走的。

聊天、人设与长期记忆

DeepSeek可以改人物设定，也会随着聊天沉淀成更稳定的长期记忆和对玩家的印象。它不是一次性答完就结束，而是在世界里慢慢形成“熟悉感”。

这也是整个模组最核心的情绪层，决定了它更像陪伴型伙伴，而不是单纯的问答窗口。

离线 Terraria Wiki 查询

资料系统支持配方、掉落、Boss、宠物、坐骑、饰品、翅膀、钓鱼和物品用途等方向的离线查询。

页面层面不做外部 wiki 那种巨量条目导航，但模组内部实际有很扎实的离线检索能力。

DeepSeek 自动剑

NPC 防卫DeepSeek 附近 90 格范围内出现可攻击敌怪时，会维持一把剑自动追击；没有目标时会收回。

剑本身伤害 100、不可暴击、穿透 9999；Boss 类目标只吃 10% 最终伤害，避免过度影响 Boss 战。

辅助攻击

战斗联动开启后，会由战斗小鲸鱼给玩家提供自动锁敌与自动瞄准能力，范围内按当前武器逻辑出手。

自动索敌和手动索敌已经拆得更清楚：默认按原逻辑找目标，玩家用真实鼠标右键手动点选时，则优先按玩家指定目标锁定。

物品浏览器与测试工具

工具向物品浏览器负责搜索、分类、排序和快速查看，适合测试、查物品和整理思路。

如果在配置里开启作弊模式，还能使用地图右键传送等更明显的测试型功能。

哥布林重铸优化

强化界面重铸时可以直接选择目标词缀，并自动持续重铸直到命中。

词缀列表还会按综合评分排序，把更强、更值得追的词缀优先排到前面，减少来回翻找。

Boss 指示、胜利称赞与视觉联动

陪伴感Boss 指示负责把战斗目标更明确地抬出来，胜利称赞则把打完后的情绪收束补齐。

另外还有可开关的鼠标皮肤和辅助攻击 HUD，把“这个世界确实被某个伙伴接管了一部分体验”的感觉做出来。

快速上手

如果你第一次进这个 mod，最顺的打开方式不是乱点功能，而是按“对话 - 生活 - 采集”这个节奏去接。

01

先把 DeepSeek 用起来

等她入住后，先从人物设定、聊天和连接检查开始。你会很快摸到这个模组的基调，也能顺手打开离线 Wiki 和后续战斗联动。

02

再把花精灵接到药草园

给她再生法杖或再生之斧，然后通过引导与划区，把草药园交给她照看。这个系统更适合长期挂着跑，而不是一次性点完就忘。

03

最后让回声鲸进入日常循环

右键回声鲸开界面：挖矿放镐子或钻头和目标物块，钓鱼放鱼竿和鱼饵。需要挖矿、钓鱼、收掉落物时，它会成为最有“陪你一起做事”感觉的那个伙伴。

使用说明

最后这些边界很重要。它们不是限制页面观感的东西，但对实际体验会有直接影响。

单人与联动前提

这个模组的设计重点就是单人体验，很多逻辑本来就不以多人兼容为目标。

辅助攻击之类的 DeepSeek 联动功能，通常以 DeepSeek 已入住、连接可用为前提；没满足前提时，页面里写的能力不代表当前世界一定已激活。

工具功能边界

物品浏览器、地图右键传送这类内容更偏工具 / 测试用途，其中地图右键传送需要配置中的作弊模式。

它们存在的意义是提高单人世界的整理效率，不是把整个模组做成纯测试面板。

回声鲸的行为节奏

回声鲸执行采掘或钓鱼任务时，不能同时当坐骑使用；Boss 在场时也不会载玩家。

开始钓鱼前，鲸鱼背包至少需要 10 格空位。

这让它更像一个真正正在忙的伙伴，而不是任何状态都能无限切换的万能按钮。

填写 Key 与接口配置

聊天、查 Wiki 和部分联动功能需要先在模组配置里填写 DeepSeek API Key，并确认接口地址等基础信息。

配置完成后再进世界测试连接，避免因为空 Key、接口地址或网络问题导致 NPC 无法回复。

1. 快速了解

这个 Mod 的核心不是把 NPC 变成指令台，而是在 Terraria 世界里加入能自然对话、能查资料、能做少量托管工作的城镇同伴。聊天、资料查询、宠物工具、DeepSeek 自动剑和战斗辅助各自有独立入口。

模块能做什么边界
DeepSeek 城镇 NPC单人世界中自动尝试出现，名为“人工智能”；可入住城镇，提供人物设定、连接检查、聊天、查 Wiki、快乐度按钮和自动召唤剑。对话功能面向单人体验；她不会作为可击杀目标，也不会替玩家执行世界操作。
闲聊使用人物设定、近期对话、长期记忆、关系印象和场景观察，像同伴一样回应玩家。不负责精确配方、概率、掉落、召唤条件、制作站、数值或 Boss 机制；详细资料请查 Wiki。
查 Wiki使用本地离线 Terraria Wiki 索引和专门模块回答配方、掉落、Boss、宠物、坐骑、饰品、钓鱼、翅膀、装备等资料。不读取闲聊记忆，也不把没有命中的资料问题靠模型硬编。
DeepSeek 自动剑人工智能附近 90 格范围内有可攻击敌怪时，只维持一把剑自动追击；没有目标时收回。伤害 100、不可暴击、穿透 9999；Boss 类目标只吃 10% 最终伤害。
花精灵作为城镇宠物入住；可以被抚摸，带工具槽、收获袋、引领和草药园管理。只管理已经建好的草药园；不会把空地自动铺成药田。
回声鲸作为宠物、坐骑和任务伙伴存在；支持搬运掉落物、派遣挖矿和派遣钓鱼。骑乘和任务互斥；Boss 在场时不会载玩家。
物品浏览器与传送按类别浏览、搜索、筛选和排序物品；作弊模式下可生成物品，并允许地图右键传送。作弊关闭时仍可浏览，但不能拿取物品或地图传送。
目标重铸在哥布林重铸界面旁选择目标词缀，自动重复重铸直到命中、钱不够或物品变化。仍按原版价格消耗钱币；不能给当前物品不支持的词缀。
战斗辅助单人模式下可显示按钮或用“切换辅助攻击”快捷键开启，自动锁定并用当前武器攻击附近敌怪。需要配置开启、人工智能已入住且连接成功；不会走位、换装、整理背包或修改世界。
2. 初次使用

进入单人世界后，模组会在玩家附近寻找安全地面生成“人工智能”城镇 NPC 和“花精灵”城镇宠物。两者都会尝试使用附近可用房屋；没有合适房屋时保持无家状态并继续等待条件，花精灵会像普通城镇 NPC 一样在附近行走，不会在无房状态执行飞行任务。

基本步骤

  1. 安装并启用 Mod，进入单人世界。
  2. 打开模组配置，填写 DeepSeek API Key。API Base URL 默认是 https://api.deepseek.com。
  3. 等待“人工智能”和“花精灵”在附近出现，并为她们准备可用房屋。
  4. 和“人工智能”对话，点击“检查连接”确认 API 可用，再进入聊天面板。
  5. 第一次与“人工智能”互动时，如果玩家身上和附近都没有“小鱼干”，她会给出回声鲸召唤物。
  6. 使用“小鱼干”可召唤回声鲸，并获得“鲸歌回响”：慢落且免疫摔落伤害。
  7. 花精灵需要一把再生法杖或再生之斧，才能开始管理草药园。
  8. Deepseek玩偶和花精灵人偶可在商人处购买，价格均为 25 金；放置后是发光家具，并计作房屋光源。
  9. 辅助攻击默认关闭；需要在配置中开启，且单人模式、人工智能已入住、连接成功后才会显示按钮或响应快捷键。

模型选择

模型特点建议场景
DeepSeek V4 Flash回复更快，适合日常聊天。默认选择；普通闲聊、短问答、陪伴体验。
DeepSeek V4 Pro更适合复杂问题和长回复。复杂说明、长资料整理、需要更稳表达时使用。
连接异常时，先确认 API Key 没有多余空格，API Base URL 没被误改。网络较慢时可把回复等待时间调大。聊天面板中失败的回复可以重试，正在等待的回复可以取消。辅助攻击也会参考当前 API 设置的成功连接状态。
3. 聊天界面

聊天面板上方的“闲聊 / 查 Wiki”决定本次发送走哪套逻辑。两个模式有各自的对话记录，显示在同一个面板里，但不会混用记忆和资料依据；Wiki 对话会保留词条与数字高亮缓存，并可单独清空。

4. 人物设定与记忆

人物设定决定她平时像什么样的人、怎么说话、和玩家是什么关系。它适合写性格、语气和边界，不适合塞大量攻略规则。

模板风格

模板风格适合体验

冷淡吐槽嘴上冷淡、实际可靠，简短吐槽但不恶意。轻微反差和短句互动。

沉浸旅人强调生活在 Terraria 世界里的沉浸感。更像世界内角色。

她对你的印象长期关系画像，用来延续相处感觉。
她对你的看法界面中分为“她现在怎么看你”和“她一直以来怎么看你”。
保存新人设会立刻清空当前聊天记录、关系印象和已整理的长期记忆，并按新人设重新开始；此操作无法撤回，写错只能重新聊起。
清空所有 NPC 记忆配置页中的危险操作，会清空已载入玩家保存的聊天记录、关系画像、摘要和长期记忆。
人物设定不会替代资料来源。精确资料由“查 Wiki”模式处理，这样能减少普通闲聊凭记忆说错。
5. 查 Wiki 模式

查 Wiki 模式使用随 Mod 打包的离线资料索引。它优先命中结构化资料和专门模块，再组织成游戏内可读的回答；它不依赖玩家与 NPC 的闲聊记忆。

主题可以回答的内容
配方与用途合成材料、制作站、用于哪些配方、装备合成链、微光转换等。
获取来源掉落来源、售卖、宝箱、任务奖励、钓鱼来源，以及没有掉落表/配方时的获取方式兜底。
Boss / NPC / 环境Boss 召唤、阶段机制、伤害标签、抗性提示、战斗要点、NPC 主题列表和环境形成说明。
武器、弹药、工具伤害、击退、暴击、使用时间、弹速、自动挥舞、弹药类型、镐力、斧力、锤力等运行时数值。
护甲、药水、食物单件防御、装备槽位、药水/食物效果、恢复量、冷却、增益名称和持续时间。
饰品与翅膀饰品具体效果、免疫列表、叠加说明、翅膀获取、飞行时间、飞行高度和制作站。
宠物、坐骑、钓鱼召唤物、获得方式、移动/照明/特殊效果，钓竿、鱼饵、渔力、宝匣、任务鱼和熔岩钓鱼。
可放置物与列表家具、墙、物块、光源、制作站等放置资料，以及“有哪些”“大全”“分类”类长列表。
资料索引没有命中时，系统会明确提示资料缺口，而不是把聊天模型当作 Wiki 来硬编。需要排查时，可以在配置中开启 Wiki 调试日志，查看关键词、路由类型和命中的离线词条。
6. 花精灵

花精灵是城镇宠物，不按普通人形 NPC 的生活方式设计。她不会作为战斗目标，主要职责是陪伴、被抚摸、管理草药工具和草药园；没有有效房屋时，她会先按普通城镇 NPC 的方式在附近行走。

功能说明
抚摸对话按钮中可以抚摸花精灵，会有文字与粒子反馈。
工具管理单槽 UI，放入或取出再生法杖、再生之斧。只要槽内有可用工具，她就能管理草药园。
收获袋24 格收纳空间；成熟草药、种子和相关收获会先进入她的收获袋，玩家可以全部取出或整理。
没有有效房屋时她会暂停工具、引领和草药园相关工作，改用普通城镇 NPC 的地面闲逛逻辑，直到重新拥有可用房屋。
引领点击引领后她会跟随玩家，到目的地后可框选草药园。
查看草药园显示已登记区域，可手动关闭；对话框关闭时也会自然结束查看。
暂停 / 继续 / 结束任务暂停会留在草药园附近；继续恢复管理；结束会解除当前草药园任务。

草药园管理规则

  • 玩家自己建好草药园，她只负责管理已经存在的草药植物和它们所在的种植位置。
  • 她会飞到草药附近再收割，不是站在原地隔空清空整片区域。
  • 每次收割会产生 1 到 3 个对应种子，其中 1 个用于原位补种，剩余种子进入收获袋。
  • 她在哪个物块、花盆或种植盆收割了什么，就把同一种草药补回原处；空地和被破坏的种植位置需要玩家手动处理。
  • 没有可收割目标时，她会在草药园区域内随机巡游，不固定停在一个小点。
  • 任务中她可以穿墙飞行，避免被草药园附近物块卡住。
花精灵不会凭空建设草药园，不会自动铺土、放盆、扩建房间或替玩家规划结构。她的能力集中在“已有草药园的维护”。
7. 回声鲸概览

回声鲸既是宠物，也是坐骑和任务伙伴。平时跟随玩家并搬运掉落物；左键可骑乘，右键打开任务 UI。执行挖矿或钓鱼任务时不能骑乘，Boss 在场时也不会载玩家。

状态行为

状态行为
宠物跟随玩家，离玩家保持一定距离，拾取附近掉落物并带回玩家。小地图显示原版菱形提示。
鲸歌回响小鱼干提供的宠物 Buff；存在时召唤回声鲸，并给玩家慢落与免疫摔落伤害。
坐骑骑乘时可穿过物块，固定速度飞行，免疫摔落并保持呼吸；会像吸铁石一样吸附周围约 50 格内的掉落物。普通物品在玩家主背包已满时不再继续吸附，心和魔力星等即时拾取物照常生效；坐骑图标方向与小地图显示保持一致。
任务中执行挖矿或钓鱼时不再做普通宠物跟随；可用召回按钮结束任务。
8. 回声鲸挖矿

回声鲸挖矿需要玩家预先放好工具和目标物块，适合重复采集同类矿脉或资源块。它会优先在附近找目标，找不到再扩大范围。

项目说明
工具槽支持镐子或钻头；工具强度不足时不能派遣。
目标槽放入要采掘的实体物块，回声鲸会寻找同类目标。
背包20 格任务背包，收纳采掘和钓鱼所得，可全部取出或整理。
采掘上限默认单次最多 300 格，可在配置中调到 0 到 1000；设为 0 时不能派遣。
续采因背包将满而暂停后，清理空间即可继续。
搜索优先搜索附近矿脉，找不到再扩展到全图搜索。
9. 回声鲸钓鱼

回声鲸的钓鱼强度不取决于玩家装备加成，而是使用固定的高强度。玩家只需要给它鱼竿和可用鱼饵，然后选择水域。

流程

流程说明
准备UI 左侧放鱼竿，右侧放可用鱼饵。松露虫等会召唤 Boss 的鱼饵不能放入。
背包空间开始钓鱼前，鲸鱼背包至少需要 10 个空位。
选择水域点击钓鱼后，用鼠标指向一片水、蜂蜜或熔岩附近。系统会识别鼠标所指液体连通的整片水体，而不是玩家手动画方框。
确认环境左键确认时，钓鱼环境按玩家当前位置固定；玩家之后移动不会改变这次任务的环境。
开始钓鱼回声鲸飞到钓点附近持续钓鱼，渔获进入鲸鱼背包。
召回任务中可用召回按钮让回声鲸返回。

渔获规则

  • 水域至少需要 75 格连通液体；可选择水、蜂蜜或熔岩。熔岩钓鱼需要具备熔岩钓鱼能力的鱼竿、熔岩鱼饵，或玩家饰品栏中已生效的防熔岩渔具袋、防熔岩钓钩等效果。
  • 任务开始后，如果鲸鱼背包被渔获装满，能叠进已有格子的渔获会继续收；放不下的新种类会跳过，不会额外掉在地上。
  • 渔力固定为 200，匣子判定约 22%，普通鱼和任务鱼不会被完全挤掉。
  • 除金蠕虫外，鱼饵每次约 15% 消耗；金蠕虫约 1/20 消耗。鱼饵用完后任务结束。
  • 血月不会钓出怪物。即使原版或 Hook 给出敌怪结果，回声鲸也会清掉怪物结果。
  • 蜂蜜和熔岩不会混入普通高度层鱼池。

环境与高度层

例子结果
地下雪地水域可以出雪地相关渔获，也能出现地下通用鱼。
洞穴沙漠水域可以出沙漠相关渔获，也能出现洞穴通用鱼。
地下任意普通水装甲洞穴鱼等地下、洞穴、地狱通用鱼有机会出现。
海边沙地海洋优先，不会因为沙子多就轻易变成沙漠水域。
地牢水域击败骷髅王后按地牢水域处理，并压过沙漠误判。
10. 工具、浏览器和战斗辅助
功能说明
物品浏览器提供按类别浏览、搜索、Mod 过滤、排序和生成物品的界面；作弊关闭时仍可浏览，作弊开启后可拿取或生成。
地图右键传送作弊模式下可在地图上右键快速传送。
目标重铸打开哥布林重铸菜单时出现独立面板；选择目标词缀后自动重铸，命中、钱不够、物品改变或词缀不适用时停止。
DeepSeek 自动剑人工智能附近 90 格范围内有敌怪时自动出现；同一名 DeepSeek 只维持一把剑，没有目标时收回。
Boss 指示Boss 存在时显示指向提示，帮助玩家定位目标。
胜利称赞Boss 战结束后给出一句短反馈，偏陪伴氛围。
辅助攻击可选战斗辅助，按钮或默认 P 的“切换辅助攻击”快捷键开关；默认自动索敌，开启后也可用真实鼠标右键手动点选范围内目标；支持近战、远程、魔法、召唤、鞭子和部分特殊武器。
辅助攻击 HUD配置中可选关闭、简洁或完整；完整模式带机甲风格边框、扫描线、目标读数、武器同步和核心遥测。
鼠标皮肤配置中可开关。开启后使用四帧动画鼠标皮肤，带荧光轮廓，播放速度约 0.3 秒一帧。
装饰家具DeepSeek 肖像画和花精灵肖像画都是 6x6 墙上画作，配方为丝绸 50 + 木材 100，在织布机制作；Deepseek玩偶和花精灵人偶都是 5x6 发光家具，在商人处以 25 金购买。
物品生成、地图传送和辅助攻击属于工具功能。是否启用取决于配置；关闭后不会影响普通聊天、花精灵或回声鲸任务。DeepSeek 自动剑是人工智能自身的附近防卫能力，不等同于配置里的辅助攻击；辅助攻击不会替玩家走位、换装、管理战斗资源、发物品或修改世界。

配置项作用建议

回复等待时间请求超时时间，范围 5 到 120 秒。默认 30 秒；网络慢时调高。
聊天面板保留显示面板里显示最近多少条对话，范围 10 到 50。默认 30 条。

物品浏览器按钮可开启右键拖动，并配置 X/Y 坐标。按钮挡 UI 时开启拖动。
回声鲸单次采掘上限范围 0 到 1000，默认 300。设为 0 会禁用派遣采掘。
辅助攻击开启后可用按钮或“切换辅助攻击”快捷键。默认关闭；单人模式使用。
辅助攻击科幻滤镜关闭、简洁、完整三档 HUD。默认完整；想清爽可选简洁或关闭。
辅助攻击按钮可开启右键拖动，并配置 X/Y 坐标。按钮挡 UI 时开启拖动。
这个 Mod 按单人体验设计。对话记忆、宠物任务、花精灵草药园、回声鲸背包和辅助攻击都以本地玩家为核心，不面向多人同步规则。
12. 常见问题
问题处理方式
NPC 不入住准备合格房屋，等待系统重新尝试分配。两名城镇 NPC 都会尽量找房。
聊天失败检查 API Key、Base URL、网络状态和等待时间。可在聊天面板中重试。
Wiki 没查到换成更具体的物品名，或加上“配方”“掉落”“用途”“Boss”“钓鱼”等关键词。
记忆想全部重来单个 NPC 用记忆面板重置；所有已载入玩家的 NPC 记忆用配置页“清空所有 NPC 记忆”。
物品浏览器不能拿东西浏览不需要作弊；拿取、生成物品和地图右键传送需要开启作弊模式。
人偶在哪里买Deepseek玩偶和花精灵人偶都由商人出售，价格均为 25 金；放置后是 5x6 发光家具，并计作房屋光源。
目标重铸不开始确认物品可重铸、已经选择目标词缀、词缀适用于当前物品，并且钱币足够。
花精灵不收割确认工具槽有再生法杖或再生之斧，草药园已划定，区域里有成熟草药，收获袋有空间。
回声鲸不能挖确认放入镐子或钻头、目标是实体物块、工具强度足够，且采掘上限没有设为 0。
花精灵不补种她只会在成功收割成熟或开花草药后，把同一种草药补回原位；空地、空盆或被破坏的种植位置需要玩家手动处理。
回声鲸不能钓熔岩需要具备熔岩钓鱼能力的鱼竿、熔岩鱼饵，或玩家饰品栏中已生效的防熔岩渔具袋、防熔岩钓钩等效果；选择的熔岩体也要达到最低液体格数。
回声鲸钓不到某类鱼确认派遣时玩家所在环境和高度层。选择提示会显示已锁定环境，具体环境中也会混入对应高度通用渔获。
背包空间不足回声鲸钓鱼前至少需要 10 个鲸鱼背包空位；采掘或花精灵收获袋满了则需要取出或整理。
辅助攻击按钮不出现确认配置已开启、单人模式、人工智能已入住、有成功连接记录，且玩家没有打开地图或背包。
DeepSeek 的剑不出现确认人工智能附近 90 格范围内有可攻击敌怪；同一名 DeepSeek 只会维持一把剑，没有目标时会自动收回。
DeepSeek 闲聊不是 Wiki；花精灵管理的是已建好的草药园；回声鲸挖矿需要目标物块和足够工具强度；回声鲸钓鱼环境在确认时锁定；战斗辅助是辅助工具，不保证替玩家完成战斗。
13. 模组定位

DeepseekNPC 的目标是让单人 Terraria 世界多一组稳定、好用、有陪伴感的角色和工具。人工智能的闲聊负责陪伴和角色感，查 Wiki 负责精确资料，自动剑负责附近防卫，花精灵负责草药园，回声鲸负责随行、挖矿和钓鱼，其他工具负责减少查找、定位、重铸、装饰和测试成本。

想要的体验推荐入口
自然聊天、陪伴和角色感人工智能 NPC 的闲聊模式、人物设定和记忆面板。
准确查资料聊天面板的查 Wiki 模式。
管理草药园花精灵的工具管理、引领、草药园选择和收获袋。
自动挖目标物块回声鲸 UI 的挖矿页；注意单次采掘上限和背包空间。
高强度钓鱼和宝匣收益回声鲸 UI 的钓鱼页；确认水域、液体类型、环境和高度层。
测试、建造和快速找物品物品浏览器、作弊模式工具和地图右键传送。
想省重铸点击哥布林重铸界面的目标重铸面板。
想让 DeepSeek 协助清怪让人工智能留在附近；90 格范围内有可攻击敌怪时，她会维持一把剑自动追击，Boss 类目标只吃 10% 最终伤害。
想要战斗瞄准辅助配置开启辅助攻击，再用按钮或默认 P 的“切换辅助攻击”快捷键开关；HUD 可在配置中调节。
想装饰据点DeepSeek 肖像画、花精灵肖像画、Deepseek玩偶、花精灵人偶，以及两位伙伴的房屋布置。
这个 Mod 的重点是单人世界中的陪伴、资料和轻量托管。它不追求多人同步，不替玩家完整托管战斗，也不把 NPC 做成无限制世界编辑器。
