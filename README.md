# DeepseekNPC

![version](https://img.shields.io/badge/version-2.1-e05d44)
![platform](https://img.shields.io/badge/platform-tModLoader-4c8eda)
![mode](https://img.shields.io/badge/mode-单人游戏-8a63d2)
![language](https://img.shields.io/badge/language-中文%20%7C%20English-45b69c)

一个以单人体验为主的 Terraria tModLoader 陪伴型模组。

模组加入了 DeepSeek、花精灵和回声鲸三位伙伴，并提供 AI 聊天与记忆、离线 Terraria Wiki 查询、药草园管理、采掘、钓鱼、物品拾取和战斗辅助等功能。
